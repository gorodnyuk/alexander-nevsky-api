Alexander Nevsky API

Alexander Nevsky API (Александр Невский API -aka- Курицын) - это бесплатный JSON API для получения коровых высказываний великого актера Александра Невского (ie. Мистер Пельменная)

Использование

GET http://nevskyapi.online/statements/random

Пример ответа:

{ "valueRu": "Абсолютли!", "valueEn": "Absolutely!" }
